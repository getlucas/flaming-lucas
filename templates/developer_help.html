Aviable API methods:
Latest API version: 2, so the basic endpoint is '/v2'
Endpoint: '/get_tokens'
Parameters: 'username' - your username (string), 'password' - your password (string)
Returns: a list of dictionaries with aviable tokens (string) and expiration time (timestamp). It is the only method that doesn't require auth_token.
Example: {"list": [ {"auth_token":"yourtoken1", "expires": 1422700651}, {...} ]}
