"flaming-robot" project
=============

Idea
----
This is a fun experiment to build a flask-based web-server 

Setup
----
in bash:
```bash
python
import flask_app as f
f.init_db()
```

Used
----
- Python 2.7 for logic, bots, etc 
- Flask 0.9 for all other (Jinja -> HTML templates) 
- SQLite3
- pythonanywhere host
